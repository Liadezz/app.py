# 🚀 Sistema de Atendimento ISP (NOC)

Sistema desenvolvido em Python com Streamlit para otimizar atendimentos técnicos em provedores de internet (ISP).

---

## 🧠 Funcionalidades

- 📋 Geração automática de registros técnicos
- 💬 Mensagens prontas para cliente
- 🤖 Sugestões inteligentes com IA
- 📊 Dashboard com métricas
- 📂 Histórico de atendimentos
- ⭐ Templates personalizados
- 🧹 Limpeza rápida de campos
- 📋 Copiar respostas com um clique

---

## 🛠️ Tecnologias

- Python
- Streamlit
- Pandas
- Groq (IA)
